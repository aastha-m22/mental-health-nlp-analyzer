# Resume Bullets

## Version A: One-Line Bullet

Built an explainable NLP classification pipeline for mental health related social media text using TF-IDF, engineered sentiment/emotion features, SMOTE, cross-validation, SHAP, and Streamlit deployment.

## Version B: Two-Line Bullet

Developed a supervised NLP pipeline for mental health signal detection using TF-IDF, Word2Vec comparison, sentiment features, emotion lexicons, linguistic pattern features, and class imbalance handling with SMOTE. Benchmarked Logistic Regression, SVM, Random Forest, XGBoost, Gradient Boosting, Complement Naive Bayes, and MLP using accuracy, precision, recall, F1, cross-validation, confusion matrices, and SHAP explanations.

## Version C: Detailed Project Description

Designed and evaluated an end-to-end machine learning system for classifying mental health related signals in social media text. The project includes advanced text preprocessing, classical NLP feature engineering, model benchmarking, cross-validation, imbalance handling, explainability, and a Streamlit inference interface. To improve technical credibility, the repository supports a real-world Dreaddit dataset workflow while retaining the synthetic dataset generator for educational demonstrations.
